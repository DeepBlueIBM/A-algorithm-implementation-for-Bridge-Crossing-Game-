Το πρόγραμμα είναι υλοποιημένο σε java. Μετά την εξαγωγή του παραδωθέντος φακέλου ανοίξτε το cmd στον φάκελο src

Για compile:
	javac *.java
Για execute:
	java Cannibals

Για παραμέτρους περάστε πρώτα τον αριθμό των κανιβάλων και των ιεραποστόλων (ο οποίος είναι ίδιος), μετά το μέγεθος της βάρκας και τέλος τον μέγιστο 
επιτρεπόμενο αριθμό διασχείσεων. Κάθε φορά εμφανίζει και το κατάλληλο μήνυμα.
Για να εκτελεστεί το προγραμμα με 3 κανιβάλους, 3 ιεραποστόλους, χωρητικότητα 2 και αριθμό διασχείσεων 20:

java Cannibals->3->2>20

Ο τρόπος που εμφανίζονται τα αποτελέσματα για το παραπάνω παράδειγμα είναι: (0,0,R,3,3) όπου τα 0 είναι ο αριθμός των κανιβάλων και των ιεραποστόλων
αντίστοιχα στην αριστερή πλευρά του ποταμού, το R είναι η θέση της βάρκας (R = δεξιά, L = αριστερά), και τα 3 είναι ο αριθμός των κανιβάλων και των 
ιεραποστόλων αντίστοιχα στην δεξιά πλευρά του ποταμού. Η βάρκα από default ξεκινάει από την αριστερή πλευρά αλλά με μια μικρή τροποποίση στην αρχική state 
μπορεί να ξεκινήσει και από τα δεξιά.
